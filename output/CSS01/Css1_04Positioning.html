CSS Floats and Positioning

Normally things in a web page show up in the order that the appear in the text.
But CSS has several ways to change the order or flow of how things are displayed on the page.
We are going to practice moving blocks around on the web page.

Floating Elements

CSS float provides the ability to have an element break free from the normal document flow and shift to the left or right as far as possible.
Once shifted, text and other inline elements can flow around it.
The amount an element can float will be determined by the size of its container and also by other floated elements in the container.
Floating Elements is a complex topic. To research more about it, look at the MDN .

Positioning Elements

In addition to floating elements, you can position them using various options.
The default positioning for elements is referred to as static . This just means that the element will be rendered in order with the normal flow of the document.
Elements can have the position attribute set to something other than static. In these cases, a few other properties become applicable; top , left , bottom , and right .

Top / Left / Bottom / Right

The top, left, bottom, and right properties can be assigned values to describe how far away from the corresponding position the box for the target element will be displayed.
For the positioning demonstration, we will use the code template below:

Relative Positioning

With Relative Positioning, the element will move a given distance from it's static placements.
position: relative;
top: 1em;
right: 1em;

Absolute Positioning

Absolute Positioning has two meanings depending on context. If the containing element has relative positioning or fixed positioning set, the child element using absolute positioning will be positioned in relation to the containers edge. However, if the containing element is not using position setting, the child element using absolute positioning will be positioned in relation to the web page's edge.
With Absolute Positioning, You can specify the distance from the elements container.
position: absolute;
top: 10px;
right: 10px;

Fixed Positioning

Have you ever seen page elements that do not move even when the page is scrolled? These elements have position:fixed applied, which causes them to be positioned relative to the browser window.
position: fixed;
top: 1em;
right: 1em;

Helpful Layout Properties

Now we will talk about some other properties that are helpful in laying out web pages.

The Display attribute.

The display attribute has options like inline and inline-block that can help control the layout of block-level elements that usually take up multiple lines.
Compare these options:

The line-height attribute.

The line-height property specifies the height of a line.
Note: Negative values are not allowed.

The z-index attribute.

The z-index property specifies the stack order of an element.
An element with greater stack order is always in front of an element with a lower stack order.
Note: z-index only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky).

Review

What does float allow us to do?
CSS float provides the ability to have an element break free from the normal document flow and shift to the left or right.
Does floating an element affect other elements in the container?
Yes. Once shifted, text and other inline elements can flow around it.
What controls the amount an element can float?
The amount an element can float will be determined by the size of its container and also by other floated elements in the container
What properties become active when the position is set to something other than static?
top , left , bottom , and right
List three position settings.
Relative, Absolute, and Fixed
What is Relative position relative to?
The elements static position.
When an element uses absolute positioning, and the containing element is using relative positioning, what does the child element position in relation to?
The containing element's edge.
What is Fixed position a fixed distance from?
The browsers edge.
How can you get a block-level element to display inline?
display: inline;
display: inline-block;
Which attribute can change the height of a line?
line-height
When elements positions cause them to stack on top of each other, which attribute specifies the stack order of the elements?
z-index
What function can help you calculate the necessary size of an element?
calc()
What function returns the value of a custom property?
var()
